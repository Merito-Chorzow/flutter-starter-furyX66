# Notes App with Weather API

A simple Flutter application for managing notes with weather information integration.

## Features

- Create notes
- Edit notes
- Delete notes
- View notes

### Native features 

- Clipboard integration

### Api integration

- Weather Api - Real-time weather data from Open-Meteo API from Warsaw, Poland (52.2296° N, 21.0122° E)

### User Experience

- Material Design
- Error handling

### To start project 

- Install Android Studio / Xcode (for emulator)
- Download and install flutter SDK  
- Install dependencies ```flutter pub get```
- Choose your targeting emulator
- Run the application ```flutter run```
